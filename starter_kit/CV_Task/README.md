# CV Task Starter Kit

Chào mừng bạn đến với **CV Task: Facial Age Regression**.

Để giúp bạn bắt đầu nhanh chóng, chúng tôi cung cấp một Jupyter Notebook mẫu trên Google Colab. Notebook này bao gồm:
- Tải và tiền xử lý dữ liệu ảnh.
- Xây dựng Dataset và Dataloader.
- Fine-tune một mô hình CNN (e.g., ResNet).
- Vòng lặp huấn luyện (Training Loop) và đánh giá (Evaluation).
- Tạo file submission hợp lệ.
