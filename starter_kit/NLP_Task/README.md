# NLP Task Starter Kit

Chào mừng bạn đến với **NLP Task: Vietnamese Social Emotion Recognition**.

Để giúp bạn bắt đầu nhanh chóng, chúng tôi cung cấp một Jupyter Notebook mẫu trên Google Colab. Notebook này bao gồm:
- Tải và tiền xử lý dữ liệu văn bản tiếng Việt.
- Tokenization và mã hóa dữ liệu.
- Fine-tune một mô hình Transformer (e.g., vinai/phobert).
- Huấn luyện và dự đoán trên tập Test.
- Tạo file submission hợp lệ.
